# Hiven

Rede social de experiências urbanas. Descubra, participe e compartilhe o que acontece na sua cidade.

## Sobre

O Hiven conecta pessoas a experiências reais perto delas. Mapa interativo, avaliações de quem realmente foi e um feed feito pra quem curte sair.

## Tech

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Mapbox GL JS
- Phosphor Icons

## Rodando

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Deploy

O projeto faz deploy automático via GitHub Pages a cada push no `master`.
